# PromptGo 产品规划需求文档

## 一、项目概述

### 1.1 项目背景
PromptGo 是一个 Prompt 提示词的汇总、搜索与上传平台，参考 aishort.top（ChatGPT Shortcut）的功能模式，旨在为用户提供高效的 AI 提示词管理与分享服务。

### 1.2 核心目标
- 构建一个简洁易用的 Prompt 管理平台
- 支持 Prompt 的展示、搜索、上传和收藏
- 提供变量模板功能，提升 Prompt 复用效率
- 开放 API 接口，支持第三方集成

### 1.3 技术栈假设
- **前端**：Web 响应式设计（HTML + Tailwind CSS）
- **后端**：RESTful API 服务
- **移动端**：优先 Web 响应式适配，原生 App 作为远期规划

### 1.4 设计规范（ui-ux-pro-max）
- **设计风格**：Flat Design（2D 极简、无阴影、干净线条）
- **主色调**：`#3B82F6`（Primary）、`#60A5FA`（Secondary）、`#F97316`（CTA）
- **字体**：Plus Jakarta Sans
- **响应式断点**：375px / 768px / 1024px / 1440px
- **暗黑模式**：使用 Tailwind `dark:` 前缀，支持系统偏好检测
- **交互动效**：150-200ms ease 过渡，无复杂动画

## 二、参考网站功能分析（aishort.top）

### 2.1 页面布局分析

| 区域 | 功能描述 |
|-----|---------|
| **顶部导航栏** | Logo、搜索框、登录按钮、语言切换 |
| **侧边分类栏** | 20+ 标签分类（写作辅助、IT/编程、生活质量等） |
| **主内容区** | Prompt 卡片瀑布流/网格布局 |
| **卡片元素** | 标题、描述摘要、标签、复制按钮 |

### 2.2 核心操作流程
```
首页浏览 → 标签筛选/关键词搜索 → 查看 Prompt 详情 → 一键复制/收藏 → 使用
                ↓
            用户登录 → 上传新 Prompt → 管理个人 Prompt
```

### 2.3 用户体验亮点
- ✅ 标签筛选与关键词搜索联动，定位效率高
- ✅ 一键复制功能，操作便捷
- ✅ 中英双语支持，覆盖面广
- ✅ 卡片式展示，信息密度适中

### 2.4 可优化点
- ⚠️ 缺少 Prompt 变量模板功能（用户需手动替换占位内容）
- ⚠️ 无版本管理，迭代记录不可追溯
- ⚠️ 缺少模型适配标注（GPT-4/Claude/文心一言等）
- ⚠️ 无 API 开放能力，第三方难以集成

## 三、PromptGo 核心功能清单

### 3.1 Prompt 展示模块

#### 3.1.1 卡片式展示
- **卡片字段**：标题、描述摘要（限 100 字）、标签列表、适配模型、使用次数、收藏数
- **展示布局**：响应式网格布局（PC 4列/平板 2列/手机 1列）
- **排序方式**：最新发布、最多使用、最多收藏

#### 3.1.2 详情页
- 完整 Prompt 内容展示
- **变量模板**：高亮 `{{变量名}}` 占位符，支持表单填充后一键复制
- 作者信息、发布时间、更新时间
- 复制按钮（统计使用次数）
- 收藏按钮

### 3.2 搜索与筛选模块

#### 3.2.1 全局搜索
- 搜索范围：标题、描述、标签、Prompt 内容
- 支持关键词高亮
- 搜索历史记录（本地存储）

#### 3.2.2 分类筛选
- **一级分类**（参考 aishort.top）：
  - 写作辅助、文章/报告、IT/编程、AI、生活质量
  - 教育/学生、学术/教师、语言/翻译、SEO、工具
  - 游戏、音乐、医生、金融顾问、企业职位 等
- **标签筛选**：支持多选，筛选条件联动
- **模型筛选**：GPT-4、GPT-3.5、Claude、文心一言、通义千问 等

### 3.3 用户上传模块

#### 3.3.1 上传表单字段
| 字段 | 类型 | 必填 | 说明 |
|-----|------|-----|------|
| 标题 | 文本 | ✅ | 限 50 字符 |
| 描述 | 文本 | ✅ | 限 200 字符，用于卡片摘要 |
| Prompt 内容 | 富文本 | ✅ | 支持 `{{变量}}` 语法 |
| 分类 | 单选 | ✅ | 一级分类 |
| 标签 | 多选 | ✅ | 最多 5 个 |
| 适配模型 | 多选 | ❌ | 默认"通用" |
| 语言 | 单选 | ✅ | 中文/英文/双语 |

#### 3.3.2 变量模板功能
- 语法：`{{变量名}}`，如 `{{产品名称}}`、`{{用户输入}}`
- 上传时自动解析变量列表
- 详情页展示变量填充表单，填充后一键复制完整 Prompt

#### 3.3.3 发布流程
```
填写表单 → 预览效果 → 直接发布（无审核）→ 进入个人管理列表
```

### 3.4 用户账户系统

#### 3.4.1 登录/注册
- 注册方式：邮箱注册、第三方登录（GitHub/Google/微信）
- 登录方式：邮箱+密码、第三方授权

#### 3.4.2 个人中心
| 功能 | 说明 |
|-----|------|
| 我的收藏 | 收藏的 Prompt 列表，支持取消收藏 |
| 我上传的 | 个人上传的 Prompt 列表，支持编辑/删除 |
| 使用统计 | 我的 Prompt 被复制/收藏次数汇总 |
| 账户设置 | 修改昵称、头像、密码 |

### 3.5 API 开放能力

#### 3.5.1 公开接口
| 接口 | 方法 | 说明 |
|-----|------|------|
| `/api/prompts` | GET | 获取 Prompt 列表，支持分页、筛选 |
| `/api/prompts/{id}` | GET | 获取单个 Prompt 详情 |
| `/api/categories` | GET | 获取分类列表 |
| `/api/tags` | GET | 获取标签列表 |
| `/api/search` | GET | 搜索接口 |

#### 3.5.2 授权接口（需 API Key）
| 接口 | 方法 | 说明 |
|-----|------|------|
| `/api/prompts` | POST | 上传新 Prompt |
| `/api/prompts/{id}` | PUT | 更新 Prompt |
| `/api/prompts/{id}` | DELETE | 删除 Prompt |
| `/api/user/favorites` | GET/POST/DELETE | 管理收藏 |

### 3.6 多语言支持
- 界面语言：中文/英文切换
- Prompt 内容：支持中文/英文/双语标注
- 默认语言：跟随浏览器设置，可手动切换

## 四、交互设计规划

### 4.1 关键页面定义

| 页面 | 路由 | 核心功能 |
|-----|------|---------|
| 首页 | `/` | Prompt 列表、搜索、分类筛选 |
| 详情页 | `/prompt/{id}` | Prompt 详情、变量填充、复制 |
| 上传页 | `/upload` | 表单填写、预览、发布 |
| 登录页 | `/login` | 登录/注册 |
| 个人中心 | `/user` | 收藏、上传管理、统计 |
| API 文档页 | `/api-docs` | 接口说明、Key 申请 |

### 4.2 用户操作路径

#### 路径 1：浏览与使用
```
首页 → 分类筛选/搜索 → 浏览卡片 → 点击卡片进入详情 → 
填充变量（如有）→ 一键复制 → 使用
```

#### 路径 2：收藏 Prompt
```
详情页 → 点击收藏按钮 → 提示登录（未登录时）→ 登录后完成收藏
```

#### 路径 3：上传 Prompt
```
点击"上传"按钮 → 登录（未登录时）→ 填写表单 → 预览效果 → 发布
```

### 4.3 交互细节标注

#### 4.3.1 按钮状态
| 按钮 | 默认 | Hover | 点击后 | 禁用 |
|-----|------|-------|--------|-----|
| 复制 | 蓝色边框 | 背景加深 | 显示"已复制 ✓"，2s 后恢复 | 灰色 |
| 收藏 | 空心心形 | 放大效果 | 实心心形 + 数字+1 | — |
| 发布 | 主色填充 | 背景加深 | Loading → 成功跳转 | 灰色 |

#### 4.3.2 错误提示
- 表单验证：字段下方红色文字提示
- 网络错误：顶部 Toast 提示，3s 自动消失
- 登录过期：弹窗提示，引导重新登录

#### 4.3.3 加载动效
- 列表加载：骨架屏（Skeleton）
- 按钮提交：Loading 动画
- 页面切换：淡入淡出
