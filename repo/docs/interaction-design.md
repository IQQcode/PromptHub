# PromptGo 交互设计文档

> 本文档定义 PromptGo 平台的关键页面、用户操作路径、交互细节和响应式布局策略。

---

## 一、关键页面定义

### 1.1 页面总览

| 页面 | 路由 | 核心功能 | 登录要求 |
|-----|------|---------|---------|
| 首页 | `/` | Prompt 列表、搜索、分类筛选 | ❌ |
| 详情页 | `/prompt/{id}` | Prompt 详情、变量填充、复制、收藏 | 收藏需登录 |
| 上传页 | `/upload` | 表单填写、预览、发布 | ✅ |
| 登录页 | `/login` | 登录/注册 | ❌ |
| 个人中心 | `/user` | 收藏管理、上传管理、统计 | ✅ |
| API 文档页 | `/api-docs` | 接口说明、API Key 申请 | Key 申请需登录 |

## 二、用户操作路径

### 2.1 路径 1：浏览与使用 Prompt

```
┌─────────┐     ┌─────────────────┐     ┌─────────────┐     ┌─────────────┐
│  首页   │ ──▶ │ 分类筛选/搜索    │ ──▶ │  浏览卡片    │ ──▶ │  点击卡片   │
└─────────┘     └─────────────────┘     └─────────────┘     └─────────────┘
                                                                   │
                                                                   ▼
┌─────────────┐     ┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  粘贴使用   │ ◀── │  一键复制    │ ◀── │  填充变量    │ ◀── │  详情页     │
└─────────────┘     └─────────────┘     └─────────────┘     └─────────────┘
                          │
                          ▼
                    ┌─────────────┐
                    │ 显示"已复制"  │
                    │  2s后恢复    │
                    └─────────────┘
```

**详细步骤**：
1. 用户访问首页，浏览 Prompt 卡片列表
2. 通过左侧分类标签筛选或使用搜索框搜索
3. 点击感兴趣的卡片进入详情页
4. （如有变量）在变量填充区输入内容
5. 点击"复制 Prompt"按钮
6. 按钮显示"已复制 ✓"，2 秒后恢复
7. 粘贴到 ChatGPT/Claude 等 AI 工具使用

### 3.2 错误提示

#### 表单验证错误
- **位置**：对应输入框下方
- **样式**：红色小号文字 + ⚠️ 图标
- **触发时机**：失去焦点时验证，实时反馈

```html
<div class="form-field error">
  <label>标题 *</label>
  <input type="text" class="error" />
  <span class="error-message">⚠️ 标题不能为空</span>
</div>
```

| 字段 | 错误类型 | 提示文案 |
|-----|---------|---------|
| 标题 | 为空 | "标题不能为空" |
| | 超长 | "标题不能超过 50 个字符" |
| 描述 | 为空 | "描述不能为空" |
| | 超长 | "描述不能超过 200 个字符" |
| Prompt 内容 | 为空 | "Prompt 内容不能为空" |
| | 超长 | "Prompt 内容不能超过 5000 个字符" |
| 分类 | 未选择 | "请选择一个分类" |
| 标签 | 未选择 | "请至少选择一个标签" |
| | 过多 | "最多选择 5 个标签" |
| 邮箱 | 格式错误 | "请输入有效的邮箱地址" |
| 密码 | 太短 | "密码至少 8 个字符" |

#### 网络错误
- **位置**：页面顶部
- **样式**：Toast 样式，红色背景，白色文字
- **显示时间**：3 秒后自动消失
- **动画**：从上方滑入，淡出消失

```html
<div class="toast toast-error">
  <span class="icon">❌</span>
  <span class="message">网络错误，请稍后重试</span>
</div>
```

| 场景 | 提示文案 |
|-----|---------|
| 网络超时 | "网络超时，请检查网络连接" |
| 服务器错误 | "服务器开小差了，请稍后重试" |
| 登录过期 | "登录已过期，请重新登录" |
| 权限不足 | "您没有权限执行此操作" |

#### 登录过期弹窗
- **类型**：模态弹窗（Modal）
- **内容**：提示信息 + "重新登录"按钮 + "取消"按钮
- **行为**：点击"重新登录"跳转登录页，点击"取消"关闭弹窗

### 3.3 加载动效

#### 列表加载 - 骨架屏（Skeleton）
- **触发时机**：首次加载、切换分类、搜索时
- **样式**：灰色占位块 + 闪烁动画
- **数量**：显示 8 个骨架卡片

```html
<div class="card-skeleton">
  <div class="skeleton-title"></div>
  <div class="skeleton-desc"></div>
  <div class="skeleton-tags"></div>
</div>
```

```css
.skeleton-title, .skeleton-desc, .skeleton-tags {
  background: linear-gradient(90deg, #E5E7EB 25%, #F3F4F6 50%, #E5E7EB 75%);
  background-size: 200% 100%;
  animation: shimmer 1.5s infinite;
}
@keyframes shimmer {
  0% { background-position: 200% 0; }
  100% { background-position: -200% 0; }
}
```

#### 按钮提交 - Loading 动画
- **样式**：按钮内显示旋转 Spinner
- **文字**：替换为"处理中..."
- **状态**：按钮禁用，防止重复提交

```html
<button class="btn-submit loading" disabled>
  <span class="spinner"></span>
  <span>处理中...</span>
</button>
```

#### 页面切换 - 淡入淡出
- **动画时长**：200ms
- **缓动函数**：ease-in-out

```css
.page-enter { opacity: 0; }
.page-enter-active { opacity: 1; transition: opacity 200ms ease-in-out; }
.page-exit { opacity: 1; }
.page-exit-active { opacity: 0; transition: opacity 200ms ease-in-out; }
```

## 六、无障碍设计（A11y）

### 6.1 键盘导航
- Tab 键可依次聚焦所有交互元素
- Enter/Space 可触发按钮
- Escape 可关闭弹窗/下拉框
- 方向键可在列表中导航

### 6.2 屏幕阅读器
- 所有图标添加 `aria-label`
- 表单字段关联 `<label>`
- 状态变化使用 `aria-live` 通知

### 6.3 颜色对比
- 文字与背景对比度 ≥ 4.5:1
- 重要信息不仅依赖颜色（添加图标/文字说明）
