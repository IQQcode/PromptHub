## 🆘 遇到问题？

### 问题 1: 页面无法加载

**症状:** 打开 http://localhost:8000 显示错误

**解决方案:**
1. 确保终端中看到 "Serving HTTP" 消息
2. 检查浏览器是否在正确的 URL 上
3. 尝试刷新页面 (F5)
4. 清除浏览器缓存

### 问题 2: 暗黑模式按钮不工作

**症状:** 点击日/月按钮没有反应

**解决方案:**
1. 打开浏览器控制台 (F12)
2. 检查是否有 JavaScript 错误
3. 确保浏览器支持 CSS class 切换

### 问题 3: 路由不工作

**症状:** URL 变化但页面不更新

**解决方案:**
1. 确保 app.js 正确加载（在 Network 标签检查）
2. 检查浏览器控制台是否有错误
3. 尝试硬刷新 (Ctrl+F5)

### 问题 4: 响应式显示不正确

**症状:** 在手机宽度下显示异常

**解决方案:**
1. 检查元素是否有 Tailwind 响应式类
2. 在 DevTools 中验证视口宽度
3. 尝试不同的浏览器
