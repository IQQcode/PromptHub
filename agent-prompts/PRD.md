# Agent Prompts - 产品需求文档 (PRD)

## 项目概述

**项目名称**: Agent Prompts（AI 提示词库）
**产品类型**: SaaS 平台 + 社区
**目标用户**: AI 爱好者、提示词工程师、内容创作者
**核心价值**: 集中管理、搜索、分享、学习 AI 提示词

---

## 一、竞品分析 - aishort.top

### aishort.top 优势功能
1. **提示词库存** - 海量分类提示词
2. **快速搜索** - 秒级搜索效率
3. **一键复制** - 快速获取提示词
4. **分类体系** - 清晰的分类导航
5. **热门排序** - 基于使用热度推荐
6. **简洁UI** - 极简设计，无干扰

### 我们的优势方向
1. **更强的搜索** - 支持中英文、标签、模糊搜索
2. **用户贡献** - 社区上传、版本管理
3. **个人收藏** - 用户账号系统、收藏夹
4. **提示词编辑** - 在线编辑、预览、版本历史
5. **数据分析** - 提示词评分、使用统计
6. **分享机制** - 提示词卡片分享、嵌入

## 二、核心功能模块

### 2.1 提示词浏览 (Browse Module)

**功能描述**
- 展示所有可用提示词
- 支持多维度浏览（分类、标签、热度）
- 快速搜索和筛选

**关键特性**
- 分类导航树（AI 工具、文案创意、代码编程、学习教育等）
- 标签系统（难度、语言、类型等）
- 热度排序、最新排序、推荐排序
- 搜索框支持中英文和模糊匹配
- 提示词卡片展示（标题、描述、标签、收藏数、用户评分）

**业务指标**
- 提示词浏览次数
- 平均停留时间
- 搜索成功率

---

### 2.2 提示词详情 (Detail Module)

**功能描述**
- 展示完整的提示词内容和元数据
- 提供交互操作（复制、收藏、评分、分享）
- 显示版本历史和相关推荐

**关键特性**
- 完整提示词内容（支持 Markdown）
- 提示词作者信息、创建时间、更新时间
- 使用人数、评分（五星制）、收藏数
- 一键复制按钮
- 收藏/取消收藏按钮
- 评分和评论系统
- 版本历史（显示修改日志）
- 相似提示词推荐
- 分享到 Twitter/LinkedIn/微博
- 嵌入代码（iframe）

**业务指标**
- 复制次数
- 收藏转化率
- 平均评分
- 评论参与度

---

### 2.3 搜索和筛选 (Search Module)

**功能描述**
- 强大的搜索引擎
- 多条件组合筛选
- 搜索结果排序和相关性算法

**关键特性**
- 全文搜索（标题、描述、内容）
- 中英文支持
- 模糊匹配和拼音搜索
- 多条件筛选：
  - 分类（多选）
  - 标签（多选）
  - 难度（初级/中级/高级）
  - 语言（中文/英文）
  - 时间范围（最近7天/30天/90天）
- 排序方式：关联度、热度、最新、评分
- 搜索建议/自动完成
- 搜索历史记录

**业务指标**
- 搜索词云（热门搜索关键词）
- 搜索无结果率
- 平均搜索结果数量

---

### 2.4 提示词上传 (Upload Module)

**功能描述**
- 允许用户上传新提示词
- 支持批量上传
- 智能提示词分析和分类

**关键特性**
- 登录和身份验证
- 提示词编辑表单：
  - 标题（必填）
  - 描述（必填）
  - 内容（必填，支持 Markdown）
  - 分类选择（必填）
  - 标签添加（多选）
  - 难度设置
  - 语言选择
- 实时预览
- 模板选择（快速开始）
- 草稿保存
- 批量导入 CSV/JSON
- 提示词模板库
- 自动分类建议
- 内容检测（去重、质量检测）

**业务指标**
- 每日上传数量
- 上传成功率
- 用户贡献比例

---

### 2.5 个人中心 (User Center Module)

**功能描述**
- 用户账号管理
- 收藏、历史、贡献管理
- 个人统计和成就

**关键特性**
- 登录/注册（邮箱、微信、GitHub OAuth）
- 个人资料编辑
- 我的收藏夹管理
- 我的贡献提示词管理
- 浏览历史记录
- 个人统计数据：
  - 收藏数、贡献数、获赞数
  - 访问热度排行
- 成就徽章系统
- 设置/隐私管理

**业务指标**
- 用户留存率
- 用户活跃度
- 平均月度活跃用户 (MAU)

---

### 2.6 推荐系统 (Recommendation Module)

**功能描述**
- 个性化提示词推荐
- 基于用户行为的算法推荐

**关键特性**
- 热门推荐（基于使用人数）
- 新增推荐（最新发布）
- 个性化推荐（基于浏览历史）
- 同类推荐（基于当前浏览内容）
- 推荐理由说明
- A/B 测试支持

**业务指标**
- 推荐点击率
- 推荐转化率
- 个性化推荐准确度

---

### 2.7 评论和社交 (Social Module)

**功能描述**
- 提示词评论讨论
- 用户互动和社区氛围

**关键特性**
- 评论系统（支持嵌套回复）
- 点赞和反对
- 评论排序（热度、时间）
- 用户信誉系统
- 评论审核机制
- @提及用户
- 评论点赞通知

**业务指标**
- 平均评论数/提示词
- 社区活跃度
- 评论垃圾率

---

### 2.8 内容管理后台 (Admin Panel)

**功能描述**
- 平台管理员工具
- 内容审核和管理

**关键特性**
- 提示词审核队列
- 内容下架/删除
- 用户管理和封禁
- 评论审核
- 数据统计和报表
- 分类/标签管理
- 公告发布
- 用户投诉处理

## 三、技术栈建议

### 前端
- 框架: React 18 + Next.js 14
- UI 组件: shadcn/ui
- 样式: Tailwind CSS
- 搜索 UI: Algolia / Meilisearch
- 状态管理: TanStack Query (React Query)
- 表单: React Hook Form + Zod

### 后端
- 框架: Node.js (Express/Fastify) 或 Python (Django/FastAPI)
- 数据库: PostgreSQL + Redis
- 搜索引擎: Elasticsearch 或 Meilisearch
- 存储: AWS S3 / 阿里云 OSS
- 缓存: Redis
- 消息队列: Bull / RabbitMQ

### DevOps
- 容器: Docker + Docker Compose
- CI/CD: GitHub Actions / GitLab CI
- 部署: Vercel (前端) + AWS/阿里云 (后端)
- 监控: Sentry / DataDog

## 五、UI/UX 设计系统

### 设计风格
- **风格**: 现代极简 (Minimalism) + 玻璃态效果 (Glassmorphism)
- **对标**: Vercel, Stripe, Framer
- **特色**: 简洁、高效、专业

### 色彩系统

**主色**
- 主蓝: `#3B82F6` (Tailwind blue-500)
- 深蓝: `#1E40AF` (Tailwind blue-800)
- 浅蓝: `#DBEAFE` (Tailwind blue-100)

**背景**
- 亮色: `#FFFFFF` / `#F9FAFB` (Tailwind gray-50)
- 暗色: `#0F172A` (Tailwind slate-900)
- 卡片: `#FFFFFF` 或 `#1E293B` (Tailwind slate-800)

**语义**
- 成功: `#10B981` (Tailwind emerald-500)
- 警告: `#F59E0B` (Tailwind amber-500)
- 错误: `#EF4444` (Tailwind red-500)
- 中立: `#6B7280` (Tailwind gray-500)

### 排版系统

**字体**
- 标题: Inter 字体族 (sans-serif)
- 正文: Inter 字体族 (sans-serif)
- 代码: JetBrains Mono (monospace)

**大小**
- H1: 32px (2rem) - 页面标题
- H2: 24px (1.5rem) - 区块标题
- H3: 20px (1.25rem) - 小标题
- Body: 16px (1rem) - 正文
- Small: 14px (0.875rem) - 辅助信息
- Micro: 12px (0.75rem) - 标签、计数

**行高**: 1.5-1.75

### 间距系统
- 极小: 4px
- 小: 8px
- 中: 16px
- 大: 24px
- 极大: 32px

### 圆角
- 小圆角: 4px
- 中圆角: 8px
- 大圆角: 12px
- 胶囊形: 9999px

### 阴影
- 轻: `0 1px 2px rgba(0,0,0,0.05)`
- 中: `0 4px 6px rgba(0,0,0,0.1)`
- 重: `0 10px 15px rgba(0,0,0,0.15)`

### 动画
- 快速过渡: 150ms
- 标准过渡: 200ms
- 缓慢过渡: 300ms
- 缓动函数: ease-in-out

### 新用户流程
1. 访问首页
2. 浏览热门提示词
3. 搜索感兴趣的提示词
4. 查看提示词详情
5. 一键复制使用

7. （可选）收藏提示词
8. （可选）上传自己的提示词

1. 登录账户
2. 查看推荐提示词
3. 访问收藏夹

5. 与社区互动（评论、点赞）
6. 上传贡献新提示词
7. 查看个人统计

## 十、风险和应对

| 风险 | 可能性 | 影响 | 应对措施 |
|------|--------|------|---------|
| 内容质量参差不齐 | 高 | 中 | 实施严格审核和用户评分机制 |
| 用户增长缓慢 | 中 | 中 | 积极的内容营销和SEO策略 |
| 内容重复 | 高 | 低 | 自动去重系统和版本管理 |
| 侵权问题 | 中 | 高 | 提示词许可证系统和法律审查 |
| 服务器成本 | 中 | 中 | 使用 CDN 和缓存策略 |
