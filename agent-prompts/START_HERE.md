## 💡 使用技巧

### 💻 在 VS Code 中快速预览
1. 安装 "Live Server" 扩展
2. 右键 index.html
3. 选择 "Open with Live Server"

### 🔄 修改颜色
```html
<!-- 在 HTML 中搜索颜色类 -->
from-blue-500   ← 改为 from-red-500
to-purple-600   ← 改为 to-orange-600
```

### 📝 修改文本
```html
<!-- 直接编辑文本内容 -->
<h1>发现和分享 AI 提示词</h1> ← 修改这里
```

### 🏗️ 添加新分类
```html
<!-- 复制现有分类按钮，修改内容 -->
<button class="glass rounded-lg p-6">
  <div class="text-3xl">🆕</div>
  <div>新分类</div>
</button>
```
