# Agent Prompts - AI 提示词库平台

一个现代化的 AI 提示词汇总、搜索和上传平台。类似 aishort.top，但具有更强大的搜索、社区驱动和个人化功能。

## 📋 项目概述

**项目名称**: Agent Prompts  
**类型**: SaaS 平台 + 社区  
**目标用户**: AI 爱好者、提示词工程师、内容创作者  
**核心价值**: 集中管理、搜索、分享、学习 AI 提示词

## 🚀 快速开始

### 查看网页

1. **首页**: 打开 `index.html` 查看项目首页、功能介绍和热门提示词
2. **浏览**: 打开 `browse.html` 浏览所有提示词、搜索和筛选
3. **详情**: 打开 `prompt-detail.html` 查看单个提示词的详细信息
4. **上传**: 打开 `upload.html` 上传新的提示词

### 本地运行

由于涉及相对路径链接，建议使用本地服务器：

```bash
# 使用 Python 3
python -m http.server 8000

# 或使用 Node.js http-server
npx http-server

# 然后访问 http://localhost:8000
```

## 🎯 核心功能模块

### 1. 提示词浏览 (Browse)
- 分类导航（10+ 分类）
- 多条件筛选（分类、标签、难度、语言）
- 多维度排序（热度、最新、评分、使用量）
- 快速搜索和模糊匹配

**特点**:
- 玻璃态 UI + 现代极简设计
- 流畅的交互和过渡效果
- 响应式设计（支持移动端）

### 2. 提示词详情 (Detail)
- 完整提示词内容展示
- 一键复制功能
- 用户评分和评论系统
- 版本历史记录
- 相关提示词推荐

**功能指标**:
- 收藏数 / 评分 / 使用人数
- 用户信息和权限
- 分享到社交媒体

### 3. 提示词上传 (Upload)
- 智能表单编辑
- 实时预览
- 快速模板选择
- 分类智能建议
- 内容检测和验证

**表单字段**:
- 标题 / 描述 / 完整内容
- 分类选择 / 标签添加
- 难度级别 / 使用语言
- 许可证选择

### 4. 搜索和筛选 (Search)
- 全文搜索（标题、描述、内容）
- 中英文支持 + 拼音搜索
- 多条件组合筛选
- 搜索建议 / 自动完成
- 搜索历史记录

## 🎨 设计系统

### 风格特征
- **设计风格**: 现代极简 (Minimalism) + 玻璃态效果 (Glassmorphism)
- **对标产品**: Vercel, Stripe, Framer
- **特色**: 简洁、高效、专业

### 色彩方案

| 元素 | 颜色 | 用途 |
|------|------|------|
| 主蓝 | #3B82F6 | CTA 按钮、主要交互 |
| 深蓝 | #1E40AF | 悬停状态 |
| 浅蓝 | #DBEAFE | 背景、标签 |
| 成功 | #10B981 | 成功提示 |
| 警告 | #F59E0B | 警告信息 |
| 错误 | #EF4444 | 错误提示 |

### 排版系统

| 级别 | 大小 | 用途 |
|------|------|------|
| H1 | 32px | 页面标题 |
| H2 | 24px | 区块标题 |
| H3 | 20px | 小标题 |
| Body | 16px | 正文 |
| Small | 14px | 辅助信息 |
| Micro | 12px | 标签、计数 |

### 动画和交互
- 过渡时间: 150-300ms
- 缓动函数: ease-in-out
- 悬停效果: 卡片向上浮动 + 阴影增强
- 加载动画: 骨架屏 / Spinner

## 📊 用户旅程

### 新用户流程
1. 访问首页了解平台
2. 浏览热门提示词
3. 搜索感兴趣的提示词
4. 查看详细信息
5. 一键复制使用
6. （可选）创建账号并收藏

### 老用户流程
1. 登录账户
2. 查看个性化推荐
3. 访问收藏夹
4. 搜索或浏览新提示词
5. 社区互动（评论、点赞）
6. 上传贡献新提示词

## 🔧 技术栈

### 前端 (当前)
- **HTML5**: 语义化标签
- **Tailwind CSS**: 原子化 CSS 框架
- **JavaScript**: 原生 JS（可交互）
- **Font Awesome**: 图标库

### 前端 (推荐生产)
- **框架**: React 18 + Next.js 14
- **UI 组件**: shadcn/ui
- **样式**: Tailwind CSS
- **状态管理**: TanStack Query

### 后端 (推荐)
- **框架**: Node.js (Express/Fastify) 或 Python (Django/FastAPI)
- **数据库**: PostgreSQL + Redis
- **搜索引擎**: Elasticsearch 或 Meilisearch
- **存储**: AWS S3 / 阿里云 OSS

### DevOps (推荐)
- **容器**: Docker + Docker Compose
- **CI/CD**: GitHub Actions
- **部署**: Vercel (前端) + AWS/阿里云 (后端)

## 📝 文件说明

### PRD.md
完整的产品需求文档，包含：
- 竞品分析
- 功能模块详细说明
- 信息架构
- UI/UX 设计系统
- 用户旅程地图
- 成功指标
- 发展路线图

### 页面文件

#### index.html (首页)
展示平台概况、核心功能、分类和 CTA
- Hero 部分：吸引用户
- 功能介绍：6 个核心功能卡片
- 分类浏览：10+ 分类按钮
- 热门提示词：3 个示例卡片
- CTA 区块：引导用户开始
- 页脚：导航链接

#### browse.html (浏览页面)
提示词浏览和搜索
- 顶部搜索框
- 左侧侧边栏：分类、排序、难度筛选
- 中央内容：提示词列表、卡片展示
- 分页控制

#### prompt-detail.html (详情页面)
单个提示词的完整信息
- 面包屑导航
- 标题和基本信息
- 提示词内容（支持 Markdown）
- 复制/收藏/分享按钮
- 使用指南
- 版本历史
- 用户评分和评论
- 相关推荐

#### upload.html (上传页面)
提示词上传和编辑
- 基本信息表单（标题、描述、内容）
- 分类和标签选择
- 难度和语言选择
- 许可证选择
- 右侧侧边栏：模板、优化建议、统计
